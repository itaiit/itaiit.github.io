# 观察者模式

观察者模式的描述是：当一个对象的状态或者行为改变时，依赖于它的对象能够得到通知并进行相应的处理。观察者模式定义的是对象之间的一种一对多的依赖关系。

在观察者模式中存在着**目标对象**和**观察者**两个角色，

## 示例代码

spring boot 中的事件通知机制就是典型的观察者模式的实现。`ApplicationListener` 接口就是观察者的统一的抽象接口，当增加一个观察者的时候，就实现这个接口就可以。

事件广播器 `SimpleApplicationEventMulticaster` 中维护着一个事件集合，广播器通过 `multicastEvent` 方法将事件广播给所有的观察者。
